<p><i>MSM</i> works as wrapper around your Media server(emby, jellyfin, kodi, plex) and helps you to manage your media files, like CRUD operations also helps to manage server services without touching server. All you need is android mobile phone and media server which are connected to same network.</p><p><br><b>Features:</b></p><ul><li>works on top of ssh</li><li>CRUD options on files</li><li>TV series can be created inside new folders or can be uploaded into existing folders</li><li>Uploads run as background tasks</li><li>Server manager(live shell, saving oneline commands)</li></ul>
